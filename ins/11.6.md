**Task:** Update `PurchaseOrderController` to handle this composite action.

1.  **Create Endpoint:** `POST /api/v1/admin/purchase-orders/items/{poItemId}/receive`
2.  **Controller Method:** `receiveAndConfigure(Request $request, $poItemId)`
    * **Input Payload:**
      ```json
      {
        "variants": [
           { "sku": "TS-RED-M", "qty": 50, "landed_cost": 500, "retail_price": 800, "attributes": {"Color": "Red", "Size": "M"} },
           { "sku": "TS-BLUE-L", "qty": 50, "landed_cost": 500, "retail_price": 800, "attributes": {"Color": "Blue", "Size": "L"} }
        ],
        "product_details": {
           "description": "...",
           "gallery_images": ["path/to/img1.jpg"],
           "meta_title": "..."
        }
      }
      ```
    * **Logic:**
        1.  **Transaction Start.**
        2.  **Find PO Item:** Get the Parent Product ID from the `po_item_id`.
        3.  **Update Parent:** Update `products` table with `product_details` (desc, gallery, etc.).
        4.  **Process Variants:** Loop through `variants`:
            * Create/Update `product_variants` (SKU, Prices).
            * Link Attributes (Color/Size).
            * **Inventory:** Increment `inventory` table `quantity` for this variant.
        5.  **Update PO Item:** Increment `stocked_quantity` by the total received amount.
        6.  **Transaction Commit.**
